Программа Radmin

Radmin (Remote Administrator) — условно бесплатная программа удалённого администрирования ПК для платформы Windows, которая позволяет полноценно работать на нескольких удалённых компьютерах с помощью графического интерфейса. Кроме этого, программа позволяет передавать файлы и использовать режим голосового или текстового общения с пользователем удаленного компьютера.

Чтобы подключится к удаленному компьютеру необходимо установить на нем серверную версию программы Radmin Server и провести начальную настройку. Необходимо задать имя пользователя и пароль для подключения. После этого можно задать права для этого пользователя. Устройство с которого производится подключение использует Radmin Viewer. Обе программы можно получить на сайте разработчика:

https://www.radmin.ru/

После подключения можно использовать множество функций этой программы — отправлять текстовые сообщения, начать текстовый чат или голосовой звонок. В программе доступны такие возможности как управление мышью, просмотр экрана, подключение в режиме командной строки, передача файлов, текстовый чат, текстовое сообщение(появится в виде отдельного окна), голосовой чат, возможность отослать сочетание клавиш или буфер обмена

Плюсы программы:

-Возможность работы с несколькими подключениями

-Возможность задать разные права доступа пользователям

-Безопасность

Минусы программы:

-Данные буфера обмена между локальным и удаленным компьютером синхронизируются вручную.

-Пароль для подключения необходимо вводить каждый раз.

-В сравнении с некоторыми конкурентами, существенным ограничением применения данной программы является невозможность работы c удалённым компьютером, имеющим «серый» IP-адрес (в обход NAT).

-Telnet без поддержки русского языка.

-При подключении иногда показывает чёрный экран.
