Программа RoyalTS

Royal TS - это кроссплатформенное решение для удаленного управления, которое может помочь контролировать такие устройства, как телефоны Mac, Android и iOS, а также ПК с Windows. Это комплексное решение для удаленного управления, которое упрощает работу с документами, открытыми несколькими пользователями одновременно.

Программу можно скачать с сайта разработчика:

www.royalapps.com/ts/win/features

В этой программе управление происходит с помощью документа. Для каждого нового соединения нужно создать новый документ, в котором будут храниться все параметры подключения. Такой файл можно хранить отдельно от программы и использовать при подключении с новых устройств другими сотрудниками компании. Также его можно передавать и использовать на другом компьютере.

С помощью Royal TS можно установить удалённое подключение через различные протоколы (RDP, VNC, SSH, S/FTP и даже веб-интерфейсы). Правами доступа можно управлять и работать с ними в команде. Списком соединений также можно поделиться, не раскрывая личных данных. Для программы можно использовать плагины, например с помощью плагина можно контролировать компьютер на котором запущен TeamViewer.

Для начала работы необходимо создать новый документ. После этого в документе создается объект, для чего есть большой выбор опций: это может быть, например, какой-нибудь тип подключения. Для каждого объекта есть множество настроек, которые будут использованы. Для разных подключений можно выбрать разные значки, чтобы было удобнее ориентироваться. Бесплатная версия имеет ограничения на 1 документ и 10 подключений.
